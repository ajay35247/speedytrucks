# AP Trucking Master Specification

## Product scope
Android-first logistics platform with three role experiences:
- Driver
- Shipper
- Admin

## Launch features
- OTP login
- role-based dashboard
- load listing
- trip lifecycle
- basic location capture
- payment and settlement data model
- fraud alert data model

## Production gaps still requiring work
- provider credentials
- end-to-end OTP
- webhook verification
- background trip sync
- admin dashboard implementation
- QA automation
- CI/CD pipeline
